# California Housing ML Model

## 📌 Project Overview
This project predicts California housing prices using Machine Learning. The model is trained on housing data and can estimate house prices based on different features.

## 🚀 Features
- Data preprocessing
- Feature scaling
- Train-test split
- Machine Learning model training
- House price prediction
- Model evaluation

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 📂 Project Structure
- main.py - Main source code
- requirements.txt - Required Python libraries

## ▶️ Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

```bash
python main.py
```

## 📊 Model Evaluation
The model is evaluated using:
- Mean Squared Error (MSE)
- R² Score

## 🎯 Objective
To build a Machine Learning model that predicts California housing prices accurately using regression techniques.
